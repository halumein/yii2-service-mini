Yii2-test-module
==========


Тестовый модуль для использования в качестве учебного материала


```
composer require halumein/yii2-service "*"
```


В конфигурационный файл приложения добавить модуль test

```php
    'modules' => [
        'service-mini' => [
            'class' => 'halumein\service-mini\Module',
        ],
    ]
```
